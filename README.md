# Визуализатор графа зависимостей для Maven пакетов

## Общее описание
Инструмент для анализа и визуализации зависимостей Java/Maven пакетов. Позволяет строить графы зависимостей и выводить их в различных форматах. Реализован на Python без использования сторонних менеджеров пакетов.

## Функции и настройки

### Основные параметры:
- `--package` - имя анализируемого пакета (обязательный)
- `--repository` - URL репозитория Maven (обязательный)  
- `--version` - версия пакета (обязательный)
- `--test-mode` - режим тестового репозитория
- `--output` - имя файла для изображения графа
- `--ascii-tree` - вывод зависимостей в формате ASCII-дерева
- `--max-depth` - максимальная глубина анализа
- `--filter` - подстрока для фильтрации пакетов

### Реализованная функциональность:
- ✅ Парсинг аргументов командной строки
- ✅ Валидация входных параметров
- ✅ Загрузка POM файлов из Maven репозиториев
- ✅ Парсинг XML и извлечение зависимостей
- ✅ Вывод прямых зависимостей в консоль
- ✅ Обработка ошибок сети и данных

## Сборка и запуск

### Требования:
- Python 3.6+
- Доступ к интернету (для загрузки POM файлов)

### Запуск:
```bash
# Полный запуск с демонстрацией обоих этапов
python stage2.py --package "org.springframework:spring-core" --repository "https://repo.maven.apache.org/maven2" --version "5.3.23"

# С дополнительными параметрами
python stage2.py --package "org.springframework:spring-core" --repository "https://repo.maven.apache.org/maven2" --version "5.3.23" --output deps.png --ascii-tree --filter "spring"
