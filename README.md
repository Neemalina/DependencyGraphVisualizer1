# Визуализатор графа зависимостей для Maven пакетов

## Общее описание
Инструмент для анализа и визуализации зависимостей Java/Maven пакетов. Позволяет строить графы зависимостей и выводить их в различных форматах.

## Функции и настройки
- Анализ зависимостей Maven пакетов
- Визуализация в виде графа (PNG)
- Вывод в формате ASCII-дерева
- Фильтрация пакетов
- Ограничение глубины анализа
